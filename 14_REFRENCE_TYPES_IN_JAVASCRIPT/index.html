Introduction to DataTypes in JavaScript

There are two types of datatypes in javascript

  1. Premitive
  2. Non-Premitive

In this video we'll learn about datatype

we can classify primitive data types into 7 categories

  1. Number
  2. String
  3. boolean
  4. Undefined
  5. Null
  6. Bigint
  7. Symbol

There is only one type of data type of non-premitive data type which is called Object

Objects are furthur devided into 3 categories

  1. Object Literals
  2. Arrays
  3. Functions
