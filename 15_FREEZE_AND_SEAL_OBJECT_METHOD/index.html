Object Method in javascript

  1. object.keys(obj) → returns array of keys
  2. object.values(obj) → returns array of values
  3. Object.entries(obj) → returns array of [key,value]
  4. Object.freeze(obj) → prevent modification
  5. Object.seal(obj) → allow change but no new keys
  6. Object.assign(target, source) → copy properties
  7. structuredClone(obj) → deep copy
