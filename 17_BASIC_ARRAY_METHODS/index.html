Basic Array methods in javascript

  1. push()
    • Adds one or more elements to the end of an array.
    • Returns the new length of the array.
    • Mutates the original array.
    • Complexity: O(1) amortized (may occasionally resize memory).
  2. pop()
    • Removes the last element from an array
    • Returns the removed element.
    • Mutates the array.
    • Complexity: O(1).
  3. shift()
    • Removes the first element from an array.
    • Returns the removed element.
    • Mutates the array.
    • Complexity: O(n) (because every element shifts left).
  4. unshift()
    • Adds one or more elements to the beginning of the array.
    • Returns the new length.
    • Mutates the array.
    • Complexity: O(n) (elements shift right).
  5. concat()
    • The concat() method in JavaScript is used to merge two or more arrays (or values) into a new array
    • t does not change (mutate) the original arrays; instead, it returns a new array.
    • Syntax : let newArray = array1.concat(array2, array3, ...);
  6. indexOf()
    • The indexOf() method in JavaScript is used to find the first occurrence of a specified element in an array.
    • It returns the index (position) of the element if found, otherwise it returns -1.
    • It performs a strict comparison (===), meaning both value and type must match.
    • Syntax: array.indexOf(searchElement, fromIndex)
      • searchElement → The element you want to find.
      • fromIndex (optional) → The index to start searching from (default = 0).
  7. includes()
    • The includes() method in JavaScript checks whether an array contains a specific element
    • It returns a boolean value
      • true if the element exists.
      • false if not found.
    • Unlike indexOf(), it doesn’t return index; just a yes/no answer.
    • Syntax : array.includes(searchElement, fromIndex)
      • searchElement → the element to look for.
      • fromIndex (optional) → the position to start searching from (default = 0).
  8. reverse()
    • Reverses the order of array elements in place.
    • Returns the same array reference (mutated).
    • Complexity: O(n).
  9. sort()
    • Sorts array in place.
    • Default: converts elements to strings, compares UTF-16 code points.
    • With compareFn(a, b):
      • Return <0 → a comes before b
      • Return 0 → no change
      • Return >0 → b before a
    • Returns the mutated array.
    • Complexity: O(n log n).
  10. slic()
    • The slice() method is used to extract a portion of an array into a new array, without changing (mutating) the original one.
    • It takes start index and end index as arguments.
    • Syntax : array.slice(start, end)
      • start → index where extraction begins (inclusive).
      • end → index where extraction ends (exclusive).
      • If end is omitted, slice continues till the end of array.
      • Negative values count from the end of the array.
  11. splice()
    • General-purpose edit tool: can insert, remove, or replace elements.
    • Parameters:
      • start: index to begin.
      • deleteCount: how many elements to remove.
      • ...items: items to insert.
    • Returns an array of deleted elements
    • Mutates the array.
    • Complexity: O(n) (shifting involved).
